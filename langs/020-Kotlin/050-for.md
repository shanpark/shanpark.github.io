# Kotlin

## for

### 1. for

```
for (int inx = 0; inx < 10; inx++) {
    System.out.println(inx);
}
```

### 2. for (Iterable)

❗ 많은 언어에서 <b>in</b>을 사용하지만 Java는 <b>in</b> 대신 <b>:</b>(콜론)을 사용한다.

```
List<String> stringList = ...;

for (String str : stringList) {
    System.out.println(str);
}
```
