## if

---

### 1. if

```java
String value = "Hello";

if (value.equals("Hello")) {
    System.out.println("Equal !!");
}
```

### 2. if ~ else

```java
String value = "Hello";

if (value.isEmpty()) {
    System.out.println("Empty !!");
} else if (value.equals("Hello")) {
    System.out.println("Equal !!");
} else {
    System.out.println("Else !!");
}
```
