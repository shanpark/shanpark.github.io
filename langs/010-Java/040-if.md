# JAVA

## if

> ❗ 아래 모든 경우에 대해서 실행할 문장이 하나인 경우 블럭 표시를 위한 중괄호('{',  '}')는 생략 가능하다.

### 1. if

```java
String value = "Hello";

if (value.equals("Hello")) {
    System.out.println("Equal !!");
}
```

### 2. if ~ else

```java
String value = "Hello";

if (value.isEmpty()) {
    System.out.println("Empty !!");
} else if (value.equals("Hello")) {
    System.out.println("Equal !!");
} else {
    System.out.println("Else !!");
}
```
